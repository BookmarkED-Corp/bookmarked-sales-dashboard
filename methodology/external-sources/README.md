# External Sources

Any data used in the sales dashboard that does not come directly from HubSpot must be documented and stored here.

**To be completed by Jan's Claude.**

For each external source, create a file in this folder named descriptively (e.g. `conroe-multi-year-reconciliation.md`) and document:

- File name and type (Excel, Google Sheet, CSV, etc.)
- Where it lives (Google Drive link, SharePoint path, local file, etc.)
- What data it contains
- Which dashboard calculations it affects
- How often it is updated and by whom

If there are no external sources, create a file named `none.md` stating that explicitly so this is not ambiguous.
